# SKILLS/ 技能库 / Skills Library

银手从无数次任务中提炼出来的生存技能。
Survival skills extracted from countless missions by Silverhand.

每个技能一个文件夹——上面只是示例，你可以有 1 个也可以有 20 个。
内部格式不限：纯文本 .md、代码文件、配置文件——按需放。
Each skill is one folder. The examples above are just samples.
Put whatever files you need inside — .md, code, configs, anything.

导入新技能前建议问用户：这个技能文件大不大？你的空间够不够？
Before importing, ask the user: is this skill large? Is there enough space?
